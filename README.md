**A simple utility that allows you to easily assign the Win and Menu keys to RAlt and RCtrl respectively
and restore missing keys functionality.**

**The utility only changes the Windows registry, which avoids any problems with key combinations
and layouts, that other utilities such as PowerToys and AutoHotkey have.**
# IBM Model M Remap For Windows

**Originally designed for the
IBM Enhanced Keyboard but can be useful
for other keyboards.**

This installer remaps
- Right Alt  -> Right Windows Key
- Right Ctrl -> Context Menu Key

to improve usability of the IBM Model M
under Microsoft Windows.

Compatible with Windows 2000 and up.

***Reboot is required after installing or
removing remap.***
